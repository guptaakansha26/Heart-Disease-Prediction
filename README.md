# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Overview
This project uses Machine Learning techniques to predict the likelihood of heart disease based on patient health parameters. The model analyzes various medical attributes and helps in early disease risk assessment.

## 🚀 Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature selection and visualization
- Machine Learning model training and evaluation
- Heart disease prediction based on user data

## 📊 Dataset Features
- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise-Induced Angina
- Oldpeak
- ST Slope

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 📈 Workflow
1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis
4. Model Training
5. Model Evaluation
6. Heart Disease Prediction

## 🎯 Objective
To develop a machine learning model that can accurately predict the presence of heart disease and assist in early diagnosis.

## 📌 Results
The trained model successfully predicts heart disease risk based on patient medical data and demonstrates the potential of machine learning in healthcare applications.
